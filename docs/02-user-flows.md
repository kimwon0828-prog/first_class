# 사용자 흐름

## 학부모 흐름
1. 체험수업 리스트 진입
2. 과목/지역/연령 기준으로 탐색
3. 수업 상세 확인
4. 체험 신청 버튼 클릭
5. 로그인 또는 회원가입
6. 신청서 작성
7. 신청 완료
8. 내 신청내역에서 상태 확인

## 선생님(studio 운영) 흐름
1. 초대/수동 생성된 teacher 계정으로 로그인
2. 체험 신청함에서 신규 신청 확인
3. 신청 상세 확인 및 상태 변경
4. 일정 관리 및 체험 진행
5. 수업 운영 정보 관리

## 인증 정책 메모
- 공개 회원가입(`/auth/sign-up`)은 parent만 가능
- teacher 공개 회원가입은 허용하지 않음
- `/studio/*` 접근 주체는 teacher로 제한
